# WDK Hackathon

Monorepo for WDK Hackathon projects.

## Packages

| Package | Description |
|---------|-------------|
| [filler/wdk-mcp-toolkit](filler/wdk-mcp-toolkit) | MCP server for Tether Wallet Development Kit |

## Setup

```bash
pnpm install
```

## Scripts

```bash
pnpm test    # run tests across all packages
pnpm lint    # lint all packages
pnpm build   # build all packages
```
